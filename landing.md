# Asteraid — ТЗ лендинга

## Общая концепция

**Asteraid** — это корпоративная веб-платформа для управления телефонией на базе Asterisk PBX.
Предоставляет единый интерфейс для администрирования одного или нескольких Asterisk-серверов, управления пользователями, просмотра статистики звонков и мониторинга в реальном времени.

**Автор/Разработчик:** Profliga SRL
**Версия продукта:** 3.1 (портал) / v.2026.02.21 (сервер)
**Стек лендинга:** Next.js + Zustand + shadcn/ui + Tailwind CSS

---

## Целевая аудитория

- IT-администраторы и инженеры телефонии
- DevOps-команды в корпоративной среде
- Руководители call-центров (роль Supervisor)
- Предприятия и организации, использующие Asterisk PBX
- Компании с несколькими офисами / несколькими PBX-серверами

### 3. Основные возможности (Feature Grid)

#### 3.1 Мониторинг в реальном времени
**Тег:** Real-time Dashboard
Через WebSocket платформа постоянно получает и отображает живые метрики со всех подключённых Asterisk-станций:
- Загрузка памяти (RAM)
- Статус транков: онлайн / оффлайн / отключён
- Статус экстеншенов: онлайн / оффлайн / отключён
- Активные каналы
- Время последней перезагрузки
- График производительности (Chart.js)

Роль: **Administrator** — видит сводные виджеты по всем станциям + детализацию по каждой.

---

#### 3.2 Управление несколькими станциями (Multi-Station)
**Тег:** Multi-Server Management
Asteraid позволяет добавить любое количество Asterisk-серверов и управлять ими централизованно:
- Добавление станции по hostname, IP
- SSH-подключение (пароль или SSH-ключ)
- Настройка часового пояса и тоновых кодов (по стране)
- Режим обслуживания (Maintenance Mode) — вывод станции из работы без удаления
- Журнал операций на станции (статус: in progress / success / error)
- Копирование SSH-ключа на сервер прямо из UI

---

#### 3.3 Визуальный редактор диалпланов (Dialplan Builder)
**Тег:** No-code / Low-code
Уникальная возможность: создание маршрутизации звонков через drag-and-drop блок-схему (ReactFlow):
- Перетаскиваемые блоки: приветствие, очередь, переадресация, условие, воспроизведение аудио и др.
- Визуальное соединение блоков стрелками
- Боковая панель с доступными блоками (Sidebar)
- Генерация реального диалплана Asterisk из визуальной схемы
- Предпросмотр итогового dialplan-кода
- Возможность создания и редактирования Rules (шаблонов маршрутизации)

---

#### 3.4 История звонков (CDR)
**Тег:** Call Detail Records
Полный журнал звонков через ODBC-подключение к CDR-базе данных:
- Фильтрация по типу (входящий, исходящий, пропущенный)
- Фильтрация по датам (одиночная или диапазон)
- Поиск по номеру телефона
- Сортировка по любому полю
- Пагинация с lazy loading
- Встроенный аудио-плеер для прослушивания записей звонков
- Экспорт данных в **CSV** и **XLSX**
- Быстрый доступ к пропущенным звонкам за период

---

#### 3.5 Управление экстеншенами, транками и очередями
**Тег:** PBX Configuration
Полное управление телефонными объектами Asterisk:

**Экстеншены (Extensions):**
- Создание, редактирование, удаление
- Привязка к шаблону конфигурации
- Включение/отключение (disabled)
- Экспертная конфигурация (raw config)

**Транки (Trunks):**
- Поддержка SIP / PJSIP
- Включение/выключение транка через переключатель
- Просмотр конфигурации в модальном окне
- Кастомные поля конфигурации

**Шаблоны (Templates):**
- Шаблоны для экстеншенов и очередей
- Создание конфигурации один раз → применение к множеству объектов

**Очереди (Queues):**
- Управление call-очередями
- Привязка к шаблону

---

#### 3.6 Управление пользователями и ролями
**Тег:** User Management + RBAC
Гибкая ролевая модель:

| Роль | Возможности |
|------|-------------|
| **Administrator** | Полный доступ: станции, конфигурация, пользователи, настройки |
| **Supervisor** | Статистика звонков по периодам, управление группами |
| **User** | Просмотр своих звонков, переадресация, информация о счёте |
| **Security** | Специализированный дашборд безопасности |

Каждая роль видит свой персонализированный Dashboard при входе.

Функции управления пользователями:
- ФИО, логин, email, пароль
- Привязка телефонных номеров (несколько + основной)
- Включение в группы
- Переключение между ролями без перелогина

---

#### 3.7 Интеграция с Active Directory
**Тег:** Active Directory
Бесшовный импорт пользователей из корпоративного каталога:
- Подключение к нескольким AD-контроллерам
- Привязка групп AD к ролям Asteraid (Admin / Supervisor / User)
- Импорт пользователей с их данными
- Отображение источника пользователя (AD vs локальная БД)
- Настройка baseDN

---

#### 3.8 Система автоматических уведомлений (AWS)
**Тег:** Auto Warning System
Встроенная система автообзвона и уведомлений:
- Создание групп уведомлений с участниками и аудио-файлом
- Загрузка аудио-промптов (с конвертацией через FFmpeg)
- Привязка контекста (dialplan) к группе
- Управление списком участников (имя + номер телефона)
- Инициаторы вызовов (номера для исходящих)

---

#### 3.9 Управление Firewall
**Тег:** Network Security
Встроенный межсетевой экран на уровне станции:
- Правила allow / deny по IP-адресу или подсети (CIDR)
- Поддержка ipset (наборы сетей)
- Описание к каждому правилу
- Применение через Ansible-скрипты на сервере

---

#### 3.10 SSL-сертификаты
**Тег:** TLS / SSL
Управление сертификатами для защищённой телефонии:
- Загрузка сертификата и ключа
- Отображение метаданных: subject, issuer, срок действия, CN/SAN
- Статус валидности (isValid, hasKey)
- Привязка активного сертификата

---

#### 3.11 Веб-терминал SSH
**Тег:** Web Terminal
Полноценный SSH-терминал прямо в браузере:
- Подключение к любой добавленной станции
- Основан на xterm.js (производительный, профессиональный)
- Поддержка копирования (ClipboardAddon)
- Автоподстройка размера под блок (FitAddon)
- Настройка размера шрифта
- Fullscreen-режим
- Быстрый запуск через ссылку с параметрами (hostname, user)

#### 3.12 Редактор конфигурационных файлов Asterisk

Встроенный Monaco-редактор (как в VS Code) для прямого редактирования файлов Asterisk:

- Настройка размера шрифта

#### 3.13 Журнал действий (Logger)
**Тег:** Audit Log
Полная история всех изменений в системе:

#### 3.14 Переадресация звонков (Redirects)
**Тег:** Call Forwarding
Тонкая настройка переадресации для каждого экстеншена:
- **CFNA** — переадресация при неответе (Call Forwarding No Answer)
- **CFB** — при занятости (Call Forwarding Busy)
- **CFU** — безусловная переадресация (Call Forward Unconditional)
- **DND** — режим «не беспокоить» (Do Not Disturb)
- Настройка таймаута
- Включение/отключение

### 6. Ключевые преимущества (Benefits section)

1. **Единый портал** — управляй всеми Asterisk-серверами компании из одного места без переключения между SSH и разными интерфейсами

2. **Не нужен Asterisk-эксперт на каждый клик** — визуальный редактор диалпланов и готовые шаблоны экстеншенов/очередей снижают порог входа

3. **Real-time контроль** — видишь состояние инфраструктуры здесь и сейчас, а не через cron-скрипты

4. **Полная история** — журнал аудита фиксирует каждое изменение: кто, что, когда и как изменил

5. **Интеграция с корпоративной инфраструктурой** — Active Directory, SSL, ODBC CDR-базы

6. **Безопасность** — встроенный Firewall, TLS-сертификаты, шифрование, ролевые права

7. **SSH-терминал в браузере** — не нужно открывать PuTTY или переключаться на другой инструмент

8. **Масштабируемость** — работает с одной станцией и с кластером; добавление новых серверов в пару кликов

### 7. Секция «Как это работает» (How it works)

**Шаг 1: Установка**
Разверните Asteraid на своём сервере. Пошаговый мастер инициализации проведёт через все настройки за 5 минут.

**Шаг 2: Подключение станций**
Добавьте свои Asterisk-серверы по hostname. Asteraid подключится по SSH и начнёт мониторинг автоматически.

**Шаг 3: Настройте пользователей**
Импортируйте сотрудников из Active Directory или создайте вручную. Назначьте роли — каждый получит свой дашборд.

**Шаг 4: Управляйте телефонией**
Транки, экстеншены, очереди, диалпланы — всё доступно из браузера. Визуальный редактор маршрутизации работает без знания синтаксиса Asterisk.

### 8. FAQ (Частые вопросы)

**Работает ли Asteraid с несколькими Asterisk-серверами?**
Да, Asteraid разработан для multi-station управления. Вы добавляете любое количество серверов и управляете ими централизованно.

**Нужны ли знания Asterisk для работы с платформой?**
Базовые знания помогут, но визуальный редактор диалпланов и шаблоны конфигураций значительно снижают порог входа.

**Поддерживается ли Active Directory?**
Да, Asteraid поддерживает подключение к нескольким AD-контроллерам с маппингом групп на роли.

**Как хранятся записи звонков?**
Asteraid подключается к вашей существующей CDR-базе данных через ODBC. Записи хранятся там же, где и раньше.

**Есть ли мобильная версия?**
Интерфейс адаптивный. Основная работа рассчитана на десктоп, но ключевые разделы доступны с планшета.

**Как обеспечивается безопасность?**
JWT-аутентификация, bcrypt-хеширование паролей, ролевые права доступа, встроенный Firewall, поддержка TLS-сертификатов.
